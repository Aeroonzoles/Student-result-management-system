# Student Result Management System

A Python-based command-line application to manage and analyze student academic results.

## Features
- Add student results with marks for Math, Science, English and Computer Science
- Automatic grade calculation (A+, A, B, C, D, Fail)
- View all student records
- Search for a specific student
- Delete student records
- Generate class report with average, top student and pass/fail statistics
- Data persistence using file handling
